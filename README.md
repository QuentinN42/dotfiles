# Dotfiles

I use [ZSH](https://www.zsh.org) with [oh my zsh](https://ohmyz.sh) and [P10k](https://github.com/romkatv/powerlevel10k).
My editor is [vim](https://www.vim.org) with [SpaceVim](https://spacevim.org).
I also use [bpytop](https://github.com/aristocratos/bpytop) (a beautiful resource monitor).


I've 2 DE : [I3](https://i3wm.org) with [polybar](https://github.com/polybar/polybar) and [gnome](https://www.gnome.org).


I've also developed a lot of simple CLI tools.
